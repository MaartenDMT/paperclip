# Paperclip Create Agent Skill

Use this skill when the task is to hire or create a new agent.

## Workflow

1. Confirm company context and permissions.
2. Read adapter configuration docs.
3. Compare existing company agent configs.
4. Draft role, icon, reporting line, adapter config, runtime config, and optional `desiredSkills`.
5. Submit the hire through the Paperclip API.
